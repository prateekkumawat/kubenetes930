############  storage class maintain with nfs 

- storage class manage purpose first need to setup nfs-csi driver 

https://github.com/kubernetes-csi/csi-driver-nfs/blob/master/docs/install-csi-driver-master.md



---
apiVersion: storage.k8s.io/v1
kind: StorageClass
metadata:
  name: nfs-store
provisioner: nfs.csi.k8s.io
parameters:
  server: 172.31.11.245
  share: /dynamicstore
reclaimPolicy: Delete
volumeBindingMode: Immediate
allowVolumeExpansion: true
mountOptions:
  - nfsvers=4.1
